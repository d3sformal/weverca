Remarks
Evaluation of every expression has common characteristics. The evaluation can need context of the program. Class FlowController represent the context, that every construct in analysis can use. It contains FlowOutputSet class, that is necessary, among other things, for access to variables and creating new values, which most expressions do. The correct context must be set before every use of class by SetContext(FlowController) method. Sometimes an expression is not correct in some way. Therefore, there is "SetContext" method to create a warning. Some values ??cannot appear in the expression, i.e. type or function values. Then the exception is thrown.
