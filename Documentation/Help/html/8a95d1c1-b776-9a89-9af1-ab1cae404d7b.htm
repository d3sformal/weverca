Remarks
The class can evaluate the following unary operations:
  • Plus
  • Minus
  • LogicNegation
  • BitNegation
  • AtSign
  • Print
  • Clone
  • Int32Cast
  • DoubleCast
  • FloatCast
  • ObjectCast
  • ArrayCast
  • UnsetCast
Some unary operation are not supported because they are not yet supported in PHP
  • Int8Cast
  • Int16Cast
  • Int64Cast
  • UInt8Cast
  • UInt16Cast
  • UInt32Cast
  • UInt64Cast
  • DecimalCast
  • BinaryCast
Conversion to boolean provides BooleanConverter for this operation:
  • BoolCast
Conversion to string provides StringConverter for these operations:
  • StringCast
  • UnicodeCast
