AbstractValueVisitor
Base class for value visitor. AbstractValueVisitor has to implement all methods from IValueVisitor interface. Every visit method calls visit method for object's supertype. Last method in the hierarchy is abstract. Allows that visitor implementations does not have to implement all methods. NOTE: * Simplifies visitor implementations. * Provides functionality for adding new visit methods into IValueVisitor. * For implementers - implement VisitValue method and then just methods you need.
