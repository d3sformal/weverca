Weverca NamespacesWeverca documentation
This is the root of the namespace tree of all Weverca tool components.
Namespaces
NamespaceDescription
Weverca
Weverca namespace contains the only executable assembly of Weverca project, which calls the analysis to given codes. This namespace itself does not contain any features.
Weverca.Analysis
Weverca.Analysis gathers all of the analyses which are run over provided PHP code and also all of the tools which are used by the analyses.
Weverca.AnalysisFramework
This group of namespaces managing the analyses and a model needed for them.
Weverca.CodeMetrics
All of the code metrics are implemented in this namespace
Weverca.Common
This namespace contains a common definitions for all the projects.
Weverca.ControlFlowGraph
In this namespace there is an implementation of building control flow graph and also all of it's features.
Weverca.MemoryModels
Memory models of Weverca project are implemented in this group of namespaces.
Weverca.Output
This group of namespaces contains classes necessary for printing output of Weverca executable tool and the web. Classes implemented in this namespace should be used for implementing any other output of the analyses.
Weverca.Parsers
This namespace contains syntax parser of PHP, which transforms given PHP sorce to AST.
