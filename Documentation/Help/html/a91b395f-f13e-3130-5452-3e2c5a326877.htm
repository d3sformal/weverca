Remarks
The visitor must resolve only the right operand, left operand of a concrete type is set in a derived class. The class can evaluate the following binary operations:
  • Equal
  • Identical
  • NotEqual
  • NotIdentical
  • LessThan
  • LessThanOrEqual
  • GreaterThan
  • GreaterThanOrEqual
  • Add
  • Sub
  • Mul
  • Div
  • Mod
  • And
  • Or
  • Xor
  • BitAnd
  • BitOr
  • BitXor
  • ShiftLeft
  • ShiftRight
